# vmcilwain.github.io

Professional site for use while job hunting.

## Dependencies

### Production

None!

### Development

- Ruby 3.3.3
- Middleman 4.6.0

## Development

- `cd` into the application root directory
- run `middleman server`
- connect via a browser to `http://localhost:4567`
- run `middleman build` when you're ready to deploy
